 installation the framework Next.js:

0: T -> node -v and npm -v (to check your Node.js and npm versions).
1: T -> npx create-next-app@latest my-next-app (to create a new Next.js project).
2: T -> cd my-next-app (to navigate into the project folder).
3: T -> npm install (to install dependencies).
4: T -> npm run dev (to start the development server).
5: T -> Go to http://localhost:3000/ (to view your Next.js app).
